HOW TO EMBED A VIDEO
Steps
Step 1: Open new .html file and set up a new table.
Step 2: Locate video on the Internet, and find and copy the embed link.
Step 3: In a new table row, start a new iframe tag by typing the info in the screenshot. Enter the width and heights and the source file link.
Step 4: After the link, type title= and then put a title in quotes, as shown in the screenshot.
